# dotfiles_installer
## homebrew_install.sh
- `sh homebrew_install.sh`
- setting file for MAC
    - brew install
        - zsh, ansible, kubectx etc...
    - brew cask
        - bettertouchtool, google-chrome, slack etc...

## install.sh
- `chmod +x install.sh`
- `./install.sh`
- make dotfiles link in homedir
    - bash
    - zprezto
        - zsh etc...
